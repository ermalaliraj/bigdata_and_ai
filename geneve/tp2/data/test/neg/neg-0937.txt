Si vous n'avez pas vu le 1er, ce film (sorti directement dans le commerce sans passer par les salles obscures) pourra vous plaire... dans le cas contraire, déception assurée, vu que cette suite est une vulgaire repompe, un remake sans interêt.

Bien que ce deuxième volet reprend l'intrigue du 1er (à savoir un gars qui peut revenir dans le passé pour changer son futur), les personnages sont médiocres dans leur jeu d'acteur. Pire encore, le scénario est faiblard, dans le sens où la personnalité de chaque personnage n'est pas du tout travaillé, pas assez en tout cas pour que les différents futurs soient excitant (à vrai dire, c'est loin d'être le cas vu que les différents présents que se crée le "voyageur dans le temps" ne se recoupe pas de façon très palpitante). 

La fin elle aussi est bâclé, sans interêt.
